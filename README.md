The following repository is culmination of month long work at IIT Dharwad Under the guidance Prof. Ameer Mulla.
The task that was assigned to us was to establish a high speed communication channel between ADC and DAC tiles present on RFSoC 4x2 with some form of modulation. 
This repository includes following circuits:
1. A basic circuit that shows how to activate and use the DAC tile present on RFSoC 4x2(title : dma_dac)
2. A basic circuit that shows how to activate and use the ADC tile present on RFSoC 4x2(title : adc_dma)
3. a complete loopback that activates both ADC and DAC tiles in the same circuit(title : pure_adc_dac)
